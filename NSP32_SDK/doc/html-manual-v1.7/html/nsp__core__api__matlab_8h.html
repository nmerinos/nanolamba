Get color information of current spectrum data.

Calculate color information and returns in X, Y, Z, r, g, b, x, y, z, cct.

double *spectrum_data; // from csCalculateSpectrum() function
double *wavelength_data; // from csCalculateSpectrum() function
int spectrum_length; // from csCalculateSpectrum() function
double X, Y, Z;
double r, g, b;
double x, y, z;
double cct;
int ret_value = acCalculateColor(spectrum_data, wavelength_data, spectrum_length,
&X, &Y, &Z, &r, &g, &b, &x, &y, &z, &cct);
Parameters
spectrum- Spectrum data [IN]
wavelength- Wavelength data [IN]
spectrum_length- Spectrum length [IN]
X- Pointer to a double which the numerical value for color (X) is returned [OUT]

Z- Pointer to a double which the numerical value for color (Z) is returned [OUT]

b- Pointer to a double which the numerical value for color (B) is returned [OUT]
x- Pointer to a double which the numerical value for color (x) is returned [OUT]
y- Pointer to a double which the numerical value for color (y) is returned [OUT]
z- Pointer to a double which the numerical value for color (z) is returned [OUT]
cct- Pointer to a double which the numerical value for CCT is returned [OUT]

Calculate spectrum.

This function returns a calculated spectrum and wavelength data with input filter data.

// acquire raw filter data from device(sensor) with shutter speed 'cur_ss'

double* wavelength_data = (double*)malloc(sizeof(double)*spectrum_length);
int ret_value = csCalculateSpectrum(filter_data, cur_ss, spectrum_data, wavelength_data );
